Q2 : Ringtones not working properly after sometime..?

When the battery saver is active you may face this issues.

Solution 1 :
  • Go to Settings
  • Battery & performance
  • Choose apps
  • Select "Multiple Ringtone Setter" and set “No restrictions”
Solution 2 :
  • Go to “Settings” then select "Multiple Ringtone Setter".
  • Search then Tap/ Click on “Follow Up” Application
  • Turn on “Autostart” option and Tap “Ok”
Note : Also make sure you have granted all permissions required by the Application.

Q3 : I am hearing beep/strange sound instead of Ringtones...why..?

This happens when the added ringtone deleted externally by user.

Solution :
  • Reinstall the Application for the "hassle-free" usage

Q4 :Granted all permissions but still not working..?

Sorry for the inconvenience you have faced with our Application.Due to android's restrictions on permissions some mobile devices won't support these kind applications. With out device permission its not possible to change the ringtones. we tried a lot and also informed to the Mobile Companies.
